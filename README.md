# Yearbook 2020

This repo holds the results of a creative coding class using [p5js](https://p5js.org/) with 3rd year graphic students @lecolededesign. The project was based on pixel images transformations. The code is available on [github](https://github.com/AtelierNum/Yearbook_2020).

The technical part of the course is available [here](https://github.com/b2renger/p5js_image_alteration/blob/master/README.md)

Ce dépôt recense les rendus produits par des étudiants de @lecolededesign en 3ème année de design graphique, sur le projet Yearbook 2020. En utilisant [p5js](https://p5js.org/) il leur était demandé de produire un portrait d'eux même et une image libre. Le code est disponible sur [github](https://github.com/AtelierNum/Yearbook_2020)

Les supports de cours sont disponible [ici](https://github.com/b2renger/p5js_image_alteration/blob/master/README.md)
